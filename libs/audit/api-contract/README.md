# @app-speed/audit/api-contract

Shared audit schemas and contract types.

## Commands

Build:

```bash
pnpm exec nx run audit-api-contract:build
```

Test:

```bash
pnpm exec nx run audit-api-contract:test
```

Lint:

```bash
pnpm exec nx run audit-api-contract:lint
```

Type-check:

```bash
pnpm exec nx run audit-api-contract:typecheck
```

Effect diagnostics:

```bash
pnpm exec nx run audit-api-contract:effect:diagnostics
```
