# @app-speed/audit/contracts

Shared audit schemas and contract types.

## Commands

Build:

```bash
pnpm exec nx run audit-contracts:build
```

Test:

```bash
pnpm exec nx run audit-contracts:test
```

Lint:

```bash
pnpm exec nx run audit-contracts:lint
```

Type-check:

```bash
pnpm exec nx run audit-contracts:typecheck
```

Effect diagnostics:

```bash
pnpm exec nx run audit-contracts:effect:diagnostics
```
